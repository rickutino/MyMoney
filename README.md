## Getting started
　開発環境構築方法。
 
* ローカルにレポジトリのクローンしてください:
>    $ git clone https://github.com/rickutino/Tameru.git
* Dependencyのインストール:
>    $ yarn 
* アプリの方法:
>    $ yarn start
